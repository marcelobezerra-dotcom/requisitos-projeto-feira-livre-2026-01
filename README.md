# Requisitos - Projeto Feira Livre 2026.1

Repositório acadêmico com os artefatos de requisitos do **Sistema de Gestão das Feiras Livres de Fortaleza**.

## Objetivo do Projeto

Documentar a visão da demanda, os termos de domínio e os principais insumos de elicitação para apoiar a análise e a especificação do sistema.

## Descrição dos Arquivos

- **Requisitos/Glossario.md**: Glossário de termos, siglas e expressões do domínio de feiras livres.
- **Requisitos/Visao-demanda.md**: Documento de visão da demanda com objetivo, proposta de valor, partes interessadas e personas.
- **Requisitos/image/Visao-demanda/**: Imagens utilizadas nos artefatos de requisitos.

## Contexto Funcional

O sistema proposto atende à Secretaria Municipal, feirantes, fiscais e cidadãos, com foco em:

- cadastro e controle de feirantes e barracas;
- apoio à emissão/gestão de licenças e taxas;
- fiscalização com consulta de regularidade;
- consulta pública de feiras, feirantes e produtos.

# Requisitos

Esta pasta concentra os artefatos de Engenharia de Requisitos do projeto **Sistema de Gestao das Feiras Livres de Fortaleza**.

## Estrutura da Pasta

```text
Requisitos/
├── README.md
├── Glossario.md
├── Visao-demanda.md
├── FontesInformacao/
└── image/
    └── Visao-demanda/
```

## Artefatos

- **Glossario.md**: termos e definicoes do dominio.
- **Visao-demanda.md**: objetivo, proposta de valor, partes interessadas, personas e escopo inicial.
- **FontesInformacao/**: materiais de apoio e fontes usadas na elaboracao dos requisitos.
- **image/**: imagens utilizadas nos documentos.

## Observacoes

- Atualize o **Historico de Versoes** sempre que houver mudanca relevante nos artefatos.
- Mantenha os documentos em linguagem clara, objetiva e consistente entre si.
